AutoRange Stable AR band edge
Method. Stable side uses Curve StableSwap-NG: invariant D solved by Newton iteration, get_y for the post-swap balance, balances rate-scaled by the per-token rate before applying the invariant. AutoRange side uses constant-product on virtual reserves x_v = L/√P, y_v = L√P with band exhaustion at Pa/Pb. Both books are sized to the same total value and centered on the same rate, so the only difference is the curve shape. Pool data via the Curve API; token prices via the DefiLlama coins API.
