AutoRange Stable AR band edge break-even
Method. Stable side uses Curve/Balancer StableSwap: invariant D solved by Newton iteration, get_y for the post-swap balance, balances rate-scaled by the per-token rate. AutoRange side uses constant-product on virtual reserves x_v = L/√P, y_v = L√P with band exhaustion at Pa/Pb. Both books are sized to the same total value and centered on the same rate. All-in cost = slippage + swap fee + gas, where gas USD = units × gwei × 10⁻⁹ × ETH price; the break-even is the trade size where both books' all-in cost is equal (smaller trades favor lower gas, larger trades favor lower slippage). Pool data via the Curve and Balancer APIs; rates via on-chain getRate(); token + ETH prices and live gas via DefiLlama and a public RPC.
