Method. Both books are evaluated with the official Balancer maths library (@balancer-labs/balancer-maths), sized to the same total value and centered on the same rate. The Stable side is a Balancer StableSwap pool (balances rate-scaled by the per-token rate). The AutoRange side is Balancer's reCLAMM (readjusting concentrated-liquidity) pool — constant-product on virtual reserves x_v = L/√P, y_v = L√P with band exhaustion at Pa/Pb. All-in cost = slippage + swap fee + gas, where gas USD = units × gwei × 10⁻⁹ × ETH price; the break-even is the trade size where both books' all-in cost is equal (smaller trades favor lower gas, larger trades favor lower slippage). Pool data via the Balancer API; rates via on-chain getRate(); token + ETH prices and live gas via DefiLlama and a public RPC. All calculations use bignumber.js at full precision; values are only rounded for display.
